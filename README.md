# Excel-File-Management-System
Excel VBA-based File Management System for folder scanning, file indexing, bulk renaming, # Excel VBA File Management System

## Overview

An Excel VBA-based File Management System that automates folder scanning, file indexing, file opening, bulk renaming, and file deletion directly from Excel.

## Features

✅ Folder Mapping

✅ Automatic File Index Generation

✅ File Opening from Excel

✅ Bulk File Renaming

✅ File Deletion with Confirmation

✅ Dashboard Reporting

## Technologies Used

- Microsoft Excel
- VBA (Visual Basic for Applications)
- FileSystemObject (FSO)

## Dashboard Summary

The system automatically tracks:

- Total Files
- Total Folders
- Total Size (MB)
- Last Refresh Date

## Business Benefits

- Reduces manual file searching
- Improves document organization
- Saves operational time
- Centralized file management
